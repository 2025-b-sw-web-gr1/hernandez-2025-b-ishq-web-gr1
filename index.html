¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Utiliza etiquetas para estructurar el contenido.

Estructura básica de una página HTML

<!DOCTYPE html>
<html>
  <head>
    <title>Título de la página</title>
  </head>
  <body>
    <h1>Encabezado principal</h1>
    <p>Este es un párrafo.</p>
  </body>
</html>

La estructura básica incluye las etiquetas <html>, <head>, <title> y <body>.

Encabezados

HTML tiene seis niveles de encabezados, de <h1> a <h6>:

<h1>Encabezado 1</h1>
<h2>Encabezado 2</h2>
<h3>Encabezado 3</h3>

Párrafos

Los párrafos se definen con la etiqueta <p>:

<p>Este es un párrafo.</p>

Enlaces

Los enlaces se crean con la etiqueta <a>:

<a href="https://www.w3schools.com">Visita W3Schools</a>

Imágenes

Las imágenes se insertan con la etiqueta <img>:

<img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools">

Elementos HTML

Un elemento HTML está formado por una etiqueta de apertura, contenido y una etiqueta de cierre. Por ejemplo:

<p>Este es un párrafo</p>

Algunos elementos no tienen etiqueta de cierre, como <br> para saltos de línea:

<p>Línea 1<br>Línea 2</p>

Ejemplo de varios elementos HTML:

<h1>Mi primer encabezado</h1>
<p>Mi primer párrafo.</p>
<a href="https://www.w3schools.com">Este es un enlace</a>

Ejemplo: Encabezados HTML

Los encabezados se usan para definir títulos y subtítulos. Hay seis niveles, del más importante (<h1>) al menos importante (<h6>):

<h1>Encabezado 1</h1>
<h2>Encabezado 2</h2>
<h3>Encabezado 3</h3>
<h4>Encabezado 4</h4>
<h5>Encabezado 5</h5>
<h6>Encabezado 6</h6>

Visualmente, <h1> es el más grande y <h6> el más pequeño. Se recomienda usar solo un <h1> por página para el título principal.

Los encabezados ayudan a organizar el contenido y mejorar la accesibilidad y el SEO de la página.

Los elementos HTML pueden estar anidados, es decir, un elemento dentro de otro:

<div>
  <p>Texto dentro de un div.</p>
</div>

Recuerda que la mayoría de los elementos tienen una etiqueta de apertura y una de cierre, pero algunos son "vacíos" y solo tienen la de apertura.
